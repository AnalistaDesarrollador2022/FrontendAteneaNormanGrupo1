    Tag HTML

    El tag HTML es una etiqueta utilizada para definir la estructura y el contenido de una página web. Es una instrucción que se utiliza para indicar al navegador cómo debe mostrar el contenido en la página. <html> Aqui va todo el contenido y estructura de la pagina o documento </html>

    Tag Head

    El tag head es una sección obligatoria en un documento HTML y contiene información sobre el documento, como metadatos, enlaces a archivos externos y otros elementos que no se muestran directamente en la página web. Estos elementos incluyen el título de la página, la descripción, la información del autor y los enlaces a hojas de estilo CSS y archivos JavaScript. <head> Aqui va todo el contenido referido</head>

    Tag Title

    El tag title en HTML se utiliza para definir el título del documento en la barra de título del navegador. Es una etiqueta obligatoria que se debe incluir en el encabezado de un documento HTML. El contenido de la etiqueta title es lo que aparece en la pestaña del navegador, y también se utiliza como el título del resultado de búsqueda en los motores de búsqueda. <title> Aqui va el titulo del documento </title>
